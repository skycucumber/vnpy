1. create a vM with Ubuntu18.04, and download private key *.pem
2. config WinSCP (convert *.pem to *.ppk) to enable login with command line.
3. config Putty to create Tunnel: http://c-nergy.be/blog/?p=6020
4. VM: Set ubuntu user(default as AzureUser) password
5. VM: install desktop tools: https://docs.microsoft.com/en-us/azure/virtual-machines/linux/use-remote-desktop
5. start a putty tunnel, and login with RDP at localhost:5555.
